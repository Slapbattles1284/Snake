# Snake 7x7

Current version: `0.0.4`

A browser-based Snake game with:
- WASD / arrow-key controls
- multi-level rounds and obstacles
- hidden cheat codes and portal room
- Angel Realm mode with enemies, health, XP, and a boss

## Run locally

```bash
python3 serve_local.py
# then open http://127.0.0.1:8000
```

## Publish Free On GitHub Pages

This game is already set up for free static hosting.

### Shortest free link

The shortest stable free link is your account site:

`https://slapbattles1284.github.io/`

To use that root link, the repository name must be:

`slapbattles1284.github.io`

### Setup steps

1. On GitHub, rename the repository from `Snake` to `slapbattles1284.github.io`.
2. Go to `Settings` -> `General` -> `Danger Zone` -> `Change repository visibility`.
3. Change the repository to `Public`.
4. Go to `Settings` -> `Pages`.
5. Under `Source`, choose `GitHub Actions`.
6. Push your latest changes to the `main` branch.

If you keep the repository name as `Snake`, the site link stays longer:

`https://slapbattles1284.github.io/Snake/`

Important:
- GitHub Free can host Pages sites for free when the repository is public.
- Do not put private info in the repo, like your full name, school, address, phone number, passwords, or secret keys.
