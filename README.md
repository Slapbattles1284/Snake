# Snake 7x7

Current version: `0.0.1`

A browser-based Snake game with:
- WASD / arrow-key controls
- multi-level rounds and obstacles
- hidden cheat codes and portal room
- Angel Realm mode with enemies, health, XP, and a boss

## Run locally

```bash
python3 -m http.server 8000
# then open http://127.0.0.1:8000
```

## Versioning

- Current version is stored in `VERSION`
- Update history is stored in `CHANGELOG.md`
